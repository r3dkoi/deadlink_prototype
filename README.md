# DEADLINK — Prototype
A psychological horror visual novel about misinformation.
*CIE152 SAE University — Project 2*

---

## Story

You are a new intern at **CLEAR** — the *Content Legitimacy & Evidence Assessment Registry*. Under the supervision of your handler, **Control**, your job is to review articles and media circulating online and decide what is true and what is misinformation.

The work feels meaningful. You believe you are making a difference.

But as you dig deeper into the Archives, something doesn't add up.

## Controls

| Input | Action |
|---|---|
| Left Click / Space / Enter | Advance dialogue |
| Right Click / Escape | Open game menu |
| S | Quick save |
| L | Open load screen |

### Menu Buttons
| Button | Function |
|---|---|
| **\|\| START \|\|** | Begin a new game |
| **\|\| CONTINUE \|\|** | Load a saved game |
| **\|\| OPTIONS \|\|** | Adjust audio and display settings |
| **\|\| QUIT \|\|** | Exit the game |
| **\|\| SAVE \|\|** | Save your current progress (in-game only) |
| **\|\| HISTORY \|\|** | Review past dialogue (in-game only) |
| **\|\| RETURN TO MAIN MENU \|\|** | Return to the main menu (in-game only) |

## Endings

The demo has two endings depending on your final choice:

- **Remove** — You side with CLEAR's mission and trust the system.
- **Keep in the system** — You question whether CLEAR's work is what it claims to be.
